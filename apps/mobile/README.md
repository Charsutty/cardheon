# Mobile app

Application mobile Cardhéon.

Stack cible :

```txt
Expo
React Native
TypeScript
Expo Router
SQLite local
```

Écrans prévus :

```txt
app/index.tsx                 Accueil
app/atelier/index.tsx         Atelier de combinaison
app/collection/index.tsx      Collection
app/collection/[slug].tsx     Fiche carte
app/constellations/index.tsx  Constellations
app/profile/index.tsx         Profil
```

Ce dossier sera initialisé avec Expo lorsque la structure de données et le premier moteur de découverte seront prêts.
