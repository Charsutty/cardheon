# 00 — Contexte du projet

## Nom

Le projet s’appelle **Cardhéon**.

Le nom évoque à la fois :

- la carte comme objet de collection ;
- le Panthéon comme lieu symbolique des grandes figures ;
- l’idée d’un ensemble de personnages reliés entre eux.

## Concept

Cardhéon est un jeu mobile éducatif dans lequel le joueur collectionne des cartes de figures historiques.

Le joueur ne reçoit pas simplement les personnages. Il les découvre en combinant des cartes-outils :

- époque ;
- lieu ;
- civilisation ;
- domaine ;
- rôle ;
- événement ;
- œuvre ;
- invention ;
- concept ;
- relation ;
- personnage déjà découvert.

Exemples :

```txt
Antiquité + Égypte + Pouvoir → Cléopâtre ou Hatchepsout selon les indices
Radioactivité + Physique + France → Marie Curie
Socrate + Philosophie + Grèce → Platon
Droits civiques + États-Unis + Rosa Parks → Martin Luther King Jr.
```

## Objectif joueur

Le joueur cherche à :

1. découvrir des personnages ;
2. compléter sa collection ;
3. débloquer de nouvelles cartes-outils ;
4. compléter des constellations thématiques ;
5. comprendre les liens historiques entre les figures.

## Différence avec Little Alchemy

Cardhéon s’inspire du plaisir de combinaison, mais ne doit pas être une simple copie de Little Alchemy.

Différences clés :

- le but principal est la découverte de personnages historiques ;
- les cartes-outils servent surtout de langage d’exploration ;
- les personnages découverts deviennent eux-mêmes utiles ;
- les liens historiques sont pédagogiques ;
- le système doit permettre plusieurs chemins vers une même découverte.

## Positionnement

Cardhéon doit être :

- ludique ;
- pédagogique ;
- accessible ;
- visuellement attractif ;
- respectueux de la complexité historique ;
- extensible à de nouveaux packs et personnages.

## Cible initiale

La cible initiale est un joueur mobile curieux, potentiellement adolescent ou adulte, qui aime :

- les jeux de collection ;
- les jeux de combinaison ;
- l’histoire ;
- les cartes ;
- les expériences de progression courtes mais régulières.

## Ambition V1

La V1 doit contenir au moins :

```txt
100 personnages
250 à 400 cartes-outils
800 à 1 500 relations historiques
30 à 50 constellations
```

Ces nombres ne sont pas arbitraires : il faut assez de cartes-outils et de relations pour que les 100 personnages ne soient pas juste 100 recettes isolées.
